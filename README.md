# Bron-tot-effect-wp3
This repository contains example data and scripts for the Stoffen Prioritering & Opsporing Tool (SPOT). Tool was developed for the TKI project [Brontoteffect-wp3](www.tkiwatertechnologie.nl/projecten/van-bron-tot-effect-integrale-aanpak-van-industriele-probleemstoffen-uit-lozingen-op-het-oppervlaktewater/)
